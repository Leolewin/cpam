\begin{Exercise}[title={Quine},difficulty=9]
A \emph{Quine} is a program that prints itself.
\label{ex:quine}
\Question\label{ex:quine q1} Write a Quine in Go.
\end{Exercise}

\begin{Answer}

/* Go quine */
package main
import "fmt"
func main() {

}
